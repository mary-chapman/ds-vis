A stack is a data structure where the last element removed from the set is element that was last inserted (First in, first out or FIFO).

setup:

  • a property (usually an array) that will hold the data contents. This usually has a max amount of items that it will hold.
  • a property that holds the top index. This usually starts at -1 or 0 and corresponds to the number of elements currently in the stack.
